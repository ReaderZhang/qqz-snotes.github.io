docker学习之路(一)

2020-09-21 2 min read

Docker最近是一个很火的东西，我最近也在学习Docker。Docker是一个开源的应用容器引擎，通过虚拟化让环境变得互相独立。与之相比较的有K8S，Mesos等等。就拿最近比较火的K8S来比较。以我所查的资料中他们俩最大的区别就是K8S是基于容器的集群管理平台，而docker则是虚拟化了环境，没有做到集群的效果。由于主要是讲解Docker，所以K8S就不提了。Docker核心的三个词是Build, Ship and Run。这三个词可以概括Docker是一个非常简易的东西。在我没学习Docker之前，一直都是在linux下JDK8,mysql之类的，还要去手动配置，收到网速的制裁，经过今天的学习Docker之后，就感觉配置环境十分方便。那么接下来进入主题。

由于我用的是centOS7,所以此处仅提供centOS7的安装方法，7及以下的可以去参考网上的教程。首先要

  • 确定你是centOS7通过uname -r指令可以得到系统的版本
  • 安装所需要的软件包yum install -y yum-utils device-mapper-persistent-data lvm2
  • yum -config-manager -add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo设置阿里云镜像仓库
  • 更新yum软件包索引 yum makecache fast
  • 安装docker ce yum -y install docker-ce
  • 启动docker systemctl start docker
  • 通过docker version查看命令，然后通过拉取hello world镜像，docker pull hello-world然后就会返回一个helloworld给用户，这就说明docker已经安装完了。
